### Passo-03-01: MACOS: Instalar Docker Desktop 
```t

Copie o Docker dmg para a pasta Aplicativos

https://hub.docker.com

# Fazer login no Docker Desktop 
Abra o Docker Desktop e faça login no Docker Hub
```

# Fazer login no Docker Desktop
